WebRTC SMS/MMS Integration Demo

This demo provides test functions, a GUI, and demonstrates the RESTful interface that would be used operationally for live streaming video available on an invite only basis from a telecommunicator (TC). Using an Agent511 application, the TC may invite a user by a link sent over SMS (or customer or subscriber) to share an audio and video feed with the TC. This can be simulated in this GUI by Creating an embed instance (below), creating a refer page, and then sending an invite to the MDN of the user. Upon receipt of the invite the user MAY elect to share audio and video by clicking on the link and then clicking on the Start button in the window. The TC must join the same session in order to view the video and hear the audio. The simulated TC view is shown in a new tab in the browser running this demo.
